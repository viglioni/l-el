#+title: l-mode

=l-mode= is a major mode built on top of =emacs-lisp-mode= that provides enhanced syntax highlighting for =l.el= features, including:

- Elixir-style =@doc= documentation annotations
- Special highlighting for s-expressions and escaped strings in docstrings
- Automatic activation when =l-syntax= is enabled

For complete documentation and examples, see the [[../../docs/api.org][API Reference]].
